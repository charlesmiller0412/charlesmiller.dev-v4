What is jQuery and what does it do?

CSS is a great tool when it comes to styling web pages. CSS is very easy to learn and very easy to organize. What if all front-end web development could be done in one language though? Just one file for an entire webpage (other than an image folder of course). This is where jQuery comes in to the picture. jQuery gives us the ability to do many different tasks all within the same package. This page was written using HTML, however, it was styled with jQuery (no CSS file attached). Here's what jQuery can do:

  • CSS Styling
  • JavaScript
  • Animations
  • Effects
  • Manipulation
  • Traversing
When jQuery is used for styling, you could say there isn't much difference in the amount of code, however, the styling options are limitless with jQuery. Take a look at the example to the right. The CSS example is strictly using the code above. Now if I were to code this text in jQuery I could also connect it to an event to make it do nearly anthing I want. Try it out!

What does it mean?

The $ highlighted in yellow is what calls for the jQuery CDN so the remainder of the code is understood. Then in red, within the parentheses, is the selector you are calling such as the ID or class. Lastly, in blue, we call the css method and tell it what css attributes we want included. Within the code in this page, you can also see several other methods being called such as slideDown(), slideUp(). show(), and hide(). Further information can be found at jQuery.com in the API Documentation.

A SIMPLE WALKTHROUGH

Where to begin:

Make sure you add the script linking to a CDN prior to beginning your code. Place it at the end of your HTML file above your local .js file and the closing body tag (</body>).

                            
                                <script src="https://code.jquery.com/jquery-3.5.1.min.js">

Now we are going to do a simple project using jQuery to create an animated dropdown menu. This would be most useful as a "hamburger" navigation menu in a mobile device.

Let's begin by setting up a button and an unordered list in your HTML file within a div since this will be an object. Set the buttons ID to menu and value to hidden. Then, give the unordered list (UL) an ID of nav while placing a few li elements within. 

                        
                            <div class="container">

                            <button id="menu" value="hidden">MENU</button>

                            <ul id="nav">

                            <li>Link 1</li>

                            <li>Link 2</li>

                            <li>Link 3</li>

                            </ul>

                            </div>

Now that you have created your menu button and list, lets add some style with jQuery in your .js file or within <script> tags. For the menu ID, we will add a width of 302px, height of 50px, as well as a background and font color of your choice. For the nav ID, add a padding of 0, margin of 0, display as none, a width of 300px, and feel free to add a background color and text alignment of your choice. Last for styling is the nav ID's li elements. Make the list style type none and give it a solid 1px border in the color of your choice. Remember the $ character is what calls the jQuery CDN, then the selector, then the method. 

                        
                            $("#menu").css({"width" : "302px", "height" : "50px", "background-color" : "#4C86DE", "color" : "#FFF"});

                            $("#nav").css({"padding" : "0", "margin" : "0","display" : "none", "background-color" : "#A5C9E0", "width" : "300px", "text-align" : "center"});

                            $("#nav li").css({"list-style-type" : "none", "border" : "1px solid #000"});

Once your code looks similar to above, we can move on to the function that will make the animation. Here are the steps to complete this final task.

  1. Create a function that initites when our #menu is clicked
  2. Add a line to initiate the slideDown() method on our #nav. You can also add the speed statement within the parentheses such as "slow"
    3. At this point you should unerstand that when the #menu is clicked the function will initiate and the #nav will slideDown().
  3. Create an if statement that if the value (hint: .val()) of #menu is equal (===) to hidden, #nav will initiate slideDown("fast") and change the value of #menu to shown
  4. Else #nav will slideUp() and change the value of #menu to hidden.

The finished function should look like this:

                        
                            $("#menu").click(function() {

                                $("#nav").slideDown("slow");
  
                                if ($("#menu").val() === "hidden") {

                                    $("#nav").slideDown("fast");

                                    $("#menu").val("shown");

                                } else {

                                    $("#nav").slideUp();

                                    $("#menu").val("hidden");

                                };

                            });
