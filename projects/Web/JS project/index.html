A BRIEF INTRODUCTION

There are many reasons you would look in to using an array to store and access data. One of them, which will be the focus of this text, is to get user data from input and retreive it. Instead of using multiple variables to store related data, you can create an array to store it. Instead of using: 

                            
                                var dog1 = "Beagle";
                                var dog2 = "Lab";
                                var dog3 = "Chihuahua";

You could use: 

                            
                                var dogs = ["Beagle", "Lab", "Chihuahua"];

I have included an example of a use of arrays. You can add as many sub-categories as you would like and then click display to see all data stored within the array.

A SIMPLE WALKTHROUGH

There are many methods available to create and alter an array. the two methods to create an array are: 

                        
                            var array = [];

                            or

                            var array = new Array();

Now that you have created an array, you can add multiple items to be stored within the array by either adding it to the code or asking for user input and using the push method or a counter.For this, we will use the push method to add the entry to the last index of the array. There should be a text input area within the HTML to gather the data with an id of userInput. 

                        
                            array = ["Data1", "Data2", "Data3"];

                            or

                            array.push(document.getElementById("userInput").value);

We now have an array with multiple pieces of data stored inside. The next thing we would want to do is access the data stored inside. We can do this by entering the index of the array we are looking to access (remember that the index starts at 0), or just enter the name of the array (which in this case is "array"). This is what it would look like inside the console. 

                        
                            array [1]
                            "Data2"

                            or

                            array
                            (3) ["Data1", "Data2", "Data3"]

Congratulations!

You have successfully created an array!
